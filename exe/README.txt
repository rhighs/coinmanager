Prima di eseguire il programma assicurarsi che nel file "dbauth.json" ci
siano le giuste credenziale per una connessione ad un DBMS Postgresql creato
seguendo il file "./other/ddl.sql".

Dopo aver eseguito il programma con successo, verrà generato un file chiamato "usersdump.txt"
quel file conterrà una lista di credenziali di tutti gli utenti presenti nel database, così da
poter effettuare il login alla "piattaforma".
